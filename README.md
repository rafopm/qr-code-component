### What I learned

Place the image inside the div and have the image inherit the rounded corners.

```html
          <div class="barcode">
            <img src="./images/image-qr-code.png" />
          </div>
```
```css
.proud-of-this-css {
  color: papayawhip;
}
```
.barcode {
  width: 290px;
  height: 290px;
  border-radius: 10px;
  overflow: hidden;
}

.barcode img {
  max-width: 100%;
  height: auto;
}
```
